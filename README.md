# Calculator App 🧮

A simple and responsive calculator built using HTML, CSS, and JavaScript.

## 💡 Features

- Basic arithmetic operations: +, –, ×, ÷, %
- Clear (`C`) and Delete (`DEL`) functions
- Smooth animations and responsive design
- Keyboard-friendly UI
